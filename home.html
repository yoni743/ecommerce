E-Commerce Web App

About This Project

Developed a full-stack e-commerce web application featuring secure user authentication, shopping cart functionality, and automated order management using n8n workflows.

Tech Stack

  • Frontend: React.js
  • Backend: Node.js (Express)
  • Database: MongoDB
  • Automation: n8n
  • Hosting: Render / Netlify

Highlights

  • Simulated 200+ concurrent users
  • RESTful API integration
  • Responsive design for mobile and desktop
